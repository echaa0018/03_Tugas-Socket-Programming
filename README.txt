how to run the program:
1. Clone the repository to your computer,
2. Open your terminal and run the server.py file by change directory to the folder in which this repository is cloned and type --python server.py--,
3. In another device (make sure you use the same network), change directory to the same folder as (2) and type --python client.py--,
4. Enter the password "luvmakima",
5. type in the server IP and port (for this use ip = <ip of the server's device in your network> and port = 9999),
6. Enter your nickname, make sure it's unique,
7. Done, go and have fun chatting!!
